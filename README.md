# Import-Multiple-SVG-Blender-3D-
With this script you can import multiple SVG'S once into Blender 3D

>How it work
- Open Blender 3D
- Go to "Scripting" tab
- Hit "New"
- Paste script and replace with your path
- Hit "Run"
- Done!
